# Bond Market Early Warning System

A next-level BondStats signal engine that combines sovereign yields, curve structure, real yields, liquidity stress and credit-sensitive assumptions into a single early warning framework.

## Features

- Current-data seeded defaults
- Early Warning Score
- Warning regime classification
- Signal breakdown
- Sovereign yield stress map
- Scenario matrix
- Fully responsive UI
